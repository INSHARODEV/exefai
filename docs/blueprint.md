# **App Name**: .exefai Landing Page

## Core Features:

- Responsive Layout: Ensure the landing page is fully responsive across desktop, tablet, and mobile devices, using a 12-column grid system for consistent layout.
- I18n Support: Implement language toggle for English and Arabic, including RTL layout support for Arabic.
- Animated Hero Section: Create a visually appealing hero section with animated background lines and parallax effect.
- Agent Card Stacking Animation: Implement the agent card stacking animation with keyboard accessible controls to cycle through cards.
- Scroll Reveal Animations: Add fade and translate animations for major blocks on scroll with staggered timing.
- Interactive Buttons: Implement micro lift and gradient hover animations for buttons.
- Content Management System Integration: The copy should be editable in the Firebase Studio content editor.

## Style Guidelines:

- Background color: Dark gray (#202326) for a modern, sophisticated look.
- Primary text color: Light gray (#EEEEEE) for readability and contrast.
- Primary brand color: Teal (#4D9192) to convey trust and innovation.
- Accent highlight gradient: Linear gradient from light cyan (#82F6F7) to teal (#4D9192) for emphasis.
- Font: 'Poppins' sans-serif for headings and body text, with weights 300, 400, 600, and 800. Note: currently only Google Fonts are supported.
- Use a 12-column grid system with 80px side margins and 20px gutter for consistent layout across devices.
- Implement subtle micro animations for buttons, scroll reveal effects, and animated background lines.
